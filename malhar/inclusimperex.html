यूनीवर्सिटी आॅफ़ मिशिगन

Exercise on inclusive imperatives.

                Use the subjunctive with the phrases in parentheses to complete short conversations between two persons. Examples:

          A:      मुन्नी अाज कितनी सुन्दर लग रही है !
             ( उसका फ़ाेटाे खिंचना )
              ( खींचना )
          B:      उसका फ़ाेटाे खिंचे ?     A:    हाँ,  खींचा जाए।

          A:      लगता है ,  अाज पानी बरसकर ही रहेगा ।     (    छाता लेना )     (  जल्दी घर लाैटना )

          B:      ताे छाता लिया जाए ?       A:    नहीं ,  जल्दी घर लाैटा जाए ।

          १       भई ,  पेट बड़े ज़ाेर से दर्द कर रहा है ।       (  डाक्टर काे बुलाना )       (  नाैकर से बुलवाना )

          २       कल हमें कमरा ख़ाली करना है ।       (  दूसरा कमरा तलाश करना )

              (  बीच पर ही साेना )

          ३       बाज़ार में घूमते -  घूमते थक गई हाे ?    मैं ताे थाेड़ी धूप खा गया हूँ ।

              (  अब घर जाना )      अाैर घर जाकर लेटना )

          ४       अाज मेरा अाम का मन है ।     (   बाज़ार से लेने चलना )       (  अपने पेड़ का ही चूसना )

          ५      ख़बरदार !      इन फूलाें में कांटे हैं ।   (  क़ैंची से कटना )     (  अाैर किसी क़िस्म के ताेड़ना )

          ६       मुझे जाने क्याें इतनी भूख लगी है ।     (  जलेबी जैसा कुछ मीटा बन जाना )

              (  कबाब बनाना )

          ७       इस महीने घर का ख़र्च बहुत ज़्यादा हाे गया है ।     (  बच्चाें काे खिलाना -  पिलाना बन्द करना )     (  तुम्हारा साड़ियाँ ख़रीदना बन्द कर देना )

          ८       देखा ?    इस बीच पर हमारे अलावा काेई भी नहीं !   (   कपड़े उतारकर तैरना )   (  समुद्र की लहराें में खेलना )
