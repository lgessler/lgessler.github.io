Exercise on  बनाए न बने: Paired verbs in the expression of incapacity

        Convert the paired clauses into single (but more complex) clauses using some form of the  बनाए न बने-construction.  Models:
  A.  हमें कोई दबाए;  हम नहीं दबेंगे।         =>
         Aa.  हम किसीसे (or  किसीके )  दबाए नहीं दबेंगे।             or:
         Ab.  हम किसीसे (or  किसीके )  दबाए नहीं बनेंगे।             or:
         Ac.  हमें किसीसे (or  किसीके )  दबाए नहीं बनेगा।
  B.  मैं यह गाँठ खोलने की कोशिश करता हूँ लेकिन ( मुझसे ) नहीं खुलती।         =>
         Ba.  यह गाँठ  ( मुझसे )  खोलते नहीं खुलती।             or:
         Bb.  यह गाँठ  ( मुझसे )  खोले नहीं खुलती।                or:
         Bc.  यह गाँठ  ( मुझसे )  खोलते नहीं बनती।               or:
         Bd.  यह गाँठ  ( मुझसे )  खोले नहीं बनती।
  C.  जितना तुम सता सको,  सता लो !         =>
          Ca.  जितना तुमसे सताते बने,  सता लो !
  1.  तसवीर बनाता हूँ;  तसवीर नहीं बनती।     =>
  2.  मानों सोने के सिक्के अाकाश से बरस रहे हों।  बटोर लो,  जितना बटोर सको !     =>
  3.  सिगरेट यह ऐसी अादत है कि छोड़ने की कोशिश करने पर भी नहीं छूटती।     =>
  4.  अाज मैंने बेटी को जगाने की बहुत कोशिश की पर वह नहीं जागी।     =>
  5.  तुम्हारी बात हम भुला रहे हैं मगर वह हमें नहीं भूलती।     =>
  6.  अचला ने अपनी सहेलियों को फँसाने का प्रयत्न तो किया फिर भी वे ( उससे ) नहीं फँसीं।     =>
  7.  मैं यह फल खाने की कोशिश कर रही हूँ लेकिन इसे खा नहीं सकती।     =>
  8.  यह दरख़्त कितना मज़बूत है !  कोई काटने की कोशिश करे फिर भी यह नहीं कटेगा।     =>
  9.  उसे लग रहा था कि काटने का प्रयास वह करे न करे यह पहाड़-सा दिन नहीं कटेगा।     =>
10.  अाँसू ऐसे बहने लगे कि उससे थम नहीं सकते थे।     =>
