Model 1  हम शक्कर-मिल की चिमनी यहाँ से साफ़ देख सकते हैं।   ( नज़र अा )   =>

Model 2  पुन्नी ने उसे देखकर अौर ज़ोर से चिल्लाना शुरू किया।   ( लग )   =>
               पुन्नी उसे देखकर अौर ज़ोर से चिल्लाने लगी।
 1.  मालिक ने हमें माफ़ी दी।   ( क्षमा कर )   =>
 2.  अप्रैल में कुछ हरियाली नज़र अाती थी।   ( दिखाई दे )   =>
 3.  पंचों ने तुम्हारी दावत स्वीकार कर ली।   ( मंज़ूर कर )   =>
 4.  यह बरताव मेरी बरदाश्त के बाहर है।   ( सहन कर )   =>
 5.  इसमें कितने पैसे ख़र्च हुए ?   ( लग )   =>

 7.  यह बात मेरे ध्यान में नहीं अाई थी।   ( ग़ौर कर )   =>
 8.  उन्होंने अपना कर्तव्य निभाया।   ( फ़र्ज़ अदा कर )   =>

10. सरकार को पिछले साल अामदनी की कितनी फ़ीसदी प्राप्त हुई ?   ( वसूल कर ले )   =>
