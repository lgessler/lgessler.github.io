तुम्हारी राह देखते हुए एक कुम्भ हुअा:  Key to exercise on participles and time expressions

 १  उन लोगों को दिल्ली में रहते दो-तीन महीने हो गए थे।
 २  ( उनको )  यह मिर्ज़ाई बनवाए  ( हुए )  पाँच साल हो गए थे ( हुए थे ) ।
 ३  पुलिस को वहाँ पहुँचते ( or  पहुँचने में )  बहुत देर नहीं लगी।
 ४  हमको यह गाड़ी ख़रीदे ( हुए )  चार साल हुए ( or  हो गए ) ।
 ५  उन्हें हिन्दी का अध्यापन करते बारह साल हो गए ( हैं ) ।
 ६  तुम्हारी विभा को यहाँ से निकले ( हुए )  ( or  निकलते हुए )  पाँच मिनट भी नहीं हुए।
 ७  सारा दिन दौड़ते हो गया।         (from Premchand's  गोदान.  See context.)
 ८  हिन्दी पढ़ते एक कुम्भ हुअा।
 ९  समय जाते देर नहीं लगती।             (from Narendra Kumar Sinha's story  शानो )
