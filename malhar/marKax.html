Exercise on  मार खा and other  खा-expressions

                 Selecting from among the nouns given in parentheses use the proper form of  खा or  खिला to make the most plausible response to speaker A.
 Model 1:  A. " भीगे कपड़े कहाँ हैं ?"
                 B. " मैंने बाहर _____ _____ रखे। "    ( बल,  धूप,  सील )  =>
                 B. " मैंने बाहर धूप खाने रखे। "
 Model 2:  A. " भय्या,  यह आदमी ' चाय-पानी '  माँग रहा है।  क्या मतलब ?"
                 B. " चाहता होगा कि हम उसे _____ _____ । "     ( घूस,  ग़ुस्सा,  घूँसा )  =>
                 B. " चाहता होगा कि हम उसे घूस खिलाएँ। "
 1. A.  वह फ़हरिस्त आपको कहाँ मिली ?
     B.  बन्द अलमारी में यह पुरानी फ़ाइल ______  ______  रही थी।  फ़हरिस्त इसी में थी।   
          ( धूप,  डुबकियाँ,  धूल )  =>
 2. A.  सर तुम्हें याद कर रहे थे।  शायद कुछ बात करना चाहते हों।
     B.  क्यों ?   क्या तुमने उनसे मेरी ______  ______ ?
          ( चुग़ली,  जान,  शरम )  =>
 3. A.  मुन्ने ने नल को फिर खुला छोड़ा है।
     B.  लगता है वह आज ______  ______  पर तुला हुआ है।  
          ( हवा,  तरस,  डाँट )  =>
 4. A.  बस - पास नहीं मिल रहा।  सब जगह ढूँढा।
     B.  क्या हुआ ?  उसे ______  ______  गई क्या ?   
          ( क़सम,  ज़मीन,  चिन्ता )  =>
 5. A.  क्या ग़ुस्सैल आदमी है !  कितनी गालियाँ !
     B.  वह तो ______  ______ बिना तुम्हें नहीं छोड़ने का !   
          ( जान,  लिहाज़,  डाँट )  =>
 6. A.  ऊपर से  " टिप "  के तौर पर सौ का नोट भी माँग लिया।
     B.  कम्बख़्त कुछ लिहाज़ नहीं करते।  थोड़ी भी तो ______  नहीं  ______ है ?   
          ( रिश्वत,  चोट,  शरम )  =>
 7. A.  धनिया को जब पता चला कि होरी मर गया है,  तब उसको इस बात का बड़ा धक्का लगा।
     B.  वह ______   _____  कर गिर पड़ी।
          ( मार,  जूते,  पछाड़ )  =>
 8. A.  वह औररत जहाँ चाहे जाय।  जब ख़ावन्द की इज़्ज़त बिगाड़ने पर आ गयी है,

     B.  जब गली-गली   _____   _____ ,  तब पता चलेगा।
          ( हवा, ठोकर,  क़सम )  =>
