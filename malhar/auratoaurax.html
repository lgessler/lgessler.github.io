Exercise on  X   तो  X . . . and the elaboration of  भी 'even'

       Choose the most plausible distribution of the elements in parentheses. Translate.
Model 1. _____  तो ______,  यह पत्थर ______  भी उठा सकता है।   ( बच्चा,  पहलवान )         ==>
                 पहलवान तो पहलवान,  यह पत्थर बच्चा भी उठा सकता है।
                  'Forget about wrestlers, even a child can lift this rock.'
Model 2. _____  तो ______,  यह पत्थर ______  से भी नहीं उठेगा।   ( बच्चे,  पहलवान )         ==>
                  बच्चे तो बच्चे,  यह पत्थर पहलवान से भी नहीं उठेगा।
      'Forget about children, even a wrestler wouldn't be able to lift this rock.'
1. _______ तो _______,  _______  भी गुम !     ( गाड़ी,  सामान )         ==>
2. _______ तो _______,  मुझसे _______  को भी सोया नहीं जाता।     ( दिन,  रात )         ==>
3. _______ तो _______,  बेटी  _______  से भी लम्बी थी।     ( बाप,  माँ )         ==>
4. _______ तो _______,  उसका  _______  भी नक़ली निकला।     ( पासपॉर्ट,  वीज़ा )         ==>
5. _______ तो _______,  बाबा को _______  भी हज़म नहीं होता।     ( पराँठा,  फुलका )         ==>
6. _______ तो _______,  यह _______  के सामने भी शरमाती है।     ( देवर,  ससुर )         ==>
7. _______ तो _______,  अब तो  _______  की भी नज़ाक़त जाती रही।   ( दिल्ली,  लखनऊ )     ==>
8. _______ तो _______,  घर में _______  भी नहीं है।     ( अशरफ़ी,  चवन्नी )         ==>
9. _______ तो _______,  वह _______  का भी परहेज़ करता है।     ( अण्डे,  प्याज़ )         ==>
10. _______ तो क्या,  उसको _______  भी नहीं आ ता।     ( सादा भैरव,  अहीर भैरव )         ==>
11. _______ तो _______,  _______  ने भी नाक में लौंग पहनी हुई थी।     ( लड़के,  लड़की )         ==>
12. _______ तो _______,  हम _______  की भी शायरी नहीं समझ सकते।     ( ग़ालिब,  ज़फ़र )     ==>
