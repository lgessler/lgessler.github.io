यूनिवर्सिटी ऑफ़ मिशिगन

Key to exercise on  बनाए न बने: Paired verbs in the expression of incapacity

 1a. तसवीर  ( मुझसे )  बनाए नहीं बनती।
 1b. तसवीर  ( मुझसे )  बनाते नहीं बनती।

  2.  मानों सोने के सिक्के अाकाश से बरस रहे हों।  बटोर लो,  जितना ( तुमसे )  बटोरते बने !     =>
        [ (2) is adapted from  प्रेमचन्द's  गोदान.  See context.]

 3a.  सिगरेट यह ऐसी अादत है कि ( किसी से )  छोड़े ( or  छोड़ते )  नहीं छूटती।
 3b.  सिगरेट यह ऐसी अादत है कि ( किसी से )  छोड़े ( or  छोड़ते )  नहीं बनती।

 4a.  अाज बेटी ( मुझसे )  जगाए ( or  जगाते )  नहीं जागी।
 4b.  अाज बेटी को ( मुझसे )  जगाए ( or  जगाते )  नहीं बना।
 4c.  अाज बेटी ( मुझसे )  जगाए ( or  जगाते )  नहीं बनी।

 5a.  तुम्हारी बात ( हमसे )  भुलाए ( or  भुलाते )  नहीं भूलती।
 5b.  तुम्हारी बात ( हमसे )  भुलाए ( or  भुलाते )  नहीं बनती।

 6a.  अपनी सहेलियाँ ( अचला से )  फँसाए नहीं फँसीं।
 6b.  अचला की सहेलियाँ ( उससे )  फँसाए नहीं फँसीं।
 6c.  अपनी सहेलियों को ( अचला से )  फँसाए नहीं बना।
 6d.  अपनी सहेलियाँ ( अचला से )  फँसाए नहीं बनीं।
 6e.  अचला की सहेलियाँ ( उससे )  फँसाए नहीं बनीं।
 6f.  अचला की सहेलियों को ( उससे )  फँसाए नहीं बना।

 7a.  यह फल मुझसे खाते ( or  खाए )  नहीं बनता।
 7b.  यह फल मुझसे खाए नहीं खाया जाता।

 8a.  यह दरख़्त कितना मज़बूत है !  यह ( किसी से )  काटे ( or  काटते )  नहीं कटेगा।
 8b.  यह दरख़्त कितना मज़बूत है !  यह ( किसी से )  काटे ( or  काटते )  नहीं बनेगा।

 9a.  उसे लग रहा था कि यह पहाड़-सा दिन उससे काटे नहीं बनेगा।
 9b.  उसे लग रहा था कि यह पहाड़-सा दिन उससे काटे नहीं कटेगा।
        [ (9b) is from  रबर बैंड, a short story by  अन्विता अब्बी.  See context.]

10.  अाँसू ऐसे बहने लगे कि उससे थामे नहीं थमते थे।
