यूनीवर्सिटी अाॅफ़ मिशिगन

Review exercise on direct versus oblique nominal forms

Choose the correct alternative.

  Model:      किताब ____ में है।       Choose from:   ( कमरा,  कमरे )             Key:  कमरे

  1.   ____ मेरे मित्र हैं।     ( ये,  इन )
  2.   ____ का घर यहाँ से दूर नहीं है।     ( ये,  इन )
  3.   ____ लोग ____ दूसरे मकान मे रहते हैं।     ( वे,  उन );   ( कोई,  किसी )
  4.  तुम ____ तरफ़ क्यों जा रहे हो ?     ( वह,  उस )
  5.  नौकर ____ से बाहर निकल गया।     ( कमरा,  कमरे )
  6.   ____ साफ़ करना ____ लोगों का काम है।     ( कमरों,  कमरे );   ( यही,  इन्हीं )
  7.  अाप ____ तरफ़ से अाए हैं ?     ( क्या,  किस )
  8.  अाज मुझसे अौर तुमसे ____ लोग मिलेंगे ?.     ( कौन,  किन )
  9.  तुम ____ से पैसे लेने अाये हो ?     ( मैं,  मुझ )
 10.   ____ दिन ____ में से ____ भी अा नहीं पाया।
        ( वह,  उस );   ( वे,  उन );   ( कोई,  किसी )
