यूनिवर्सिटी ऑफ़ मिशिगन

Exercise on expressing concession with  सही

      Using common preferences, decide which items in parentheses are most plausibly put in which blanks.  Examples:

  A.  गहने _____  के नहीं,  ____  के सही।     ( चाँदी,     सोने )       =>    गहने सोने के नहीं,  चाँदी के सही।

  B.  हम ____   नहीं ख़रीद सकेंगे,  न सही,   तो _____  चलेगी ही सही।     ( साइकल,    स्कूटर )     =>
            हम स्कूटर नहीं ख़रीद सकेंगे,  न सही,   तो साइकल चलेगी ही सही।

  1.  ठीक है,  ____   नहीं है,  तो फिर ____  ही सही।         ( टैक्सी,     बैल-गाड़ी )         =>

  2.  पैसे ____   नहीं दोगे,  न सही,  ____  दे देना।         ( अाज,     कल )         =>

  3.  अच्छा,   ____  नहीं है,  न सही !  फिर थोड़ी  ____  खिलाअो तो सही।       ( जिलेबी,     बफ़ीर् )     =>

  4.   ____  नहीं तो ____  ही सही !         ( अोहाइयो इस्टेट,     मिशिगन )         =>

  5.  चलो,  ____   ही सही,  अगर ____  न सही।         ( अक़ल,     शकल )         =>

  6.   ____   नहीं तो ____  ही सही।         ( चौसा-लंगड़ा,    तोतापरी )         =>

  7.  मिलने नहीं ____  -गे,  न सही,  ____  तो सही।         ( अाएँ,     बुलाएँ )         =>

  8.  मन हुअा,  चला जाऊँ -  ____ के लिये ही सही,  छुट्टी तो मिलेगी।     ( कुछ देर,     हमेशा )     =>

  9.  फिर भी छोटे-छोटे टुकड़े जोड़ें तो उसके चरित्र का एक ____  न सही,  ____  चित्र तो सामने
    अा ही सकता है।         ( कामचलाऊ,     पूरा )         =>

10.  ज़ुल्फ़-ए-स्याह ____  की नहीं तो ____  की सही !         ( ख़िज़ाब,     शबाब )         =>
