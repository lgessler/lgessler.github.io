V-ए बिना :  Exercise 1

  Use V-ए बिना (or V-ए बग़ैर ) to express strong obligation, assertion, or determination.

 Model A:  कुछ कहना पड़ेगा।     =>     कुछ कहे बिना (  काम )  नहीं चलेगा।

 Model B:  जुलाई में ताे पानी बरसता ही।     =>     जुलाई में ताे पानी बरसे बिना नहीं रहता।

 Model C:  मैं पैसे लेकर ही जाऊँगा।     =>     मैं पैसे लिए बिना ( or  बग़ैर )  नहीं जाऊँगा।

 1.  वह ताे तेरी ख़बर लेगी ही।     =>
 2.  जाना ही पड़ेगा।     =>
 3.  इस माैसम मैं दस्ताने पहनकर ही निकलाे।     =>
 4.  उन्हें यहाँ से निकालूँगी ही।     =>
 5.  ग़ाैर करना पड़ेगा ही।     =>
 6.  ध्यान देना पड़ेगा ही।     =>
 7.  अपना काम ख़तम करके ही साेने जाऊँगा।     =>
 8.  यह ताे हाेगा ही !     =>
 9.  उनसे सलाम कहकर ही दाख़िल हाेइएगा।     =>
10.  थाेड़ा साेचना-समझना पड़ेगा।     =>
