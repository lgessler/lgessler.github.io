Exercise on uses of -वाला.

(adapted from Chapter Two of  Hindi Structures, pp 16-17)

      Convert relative clauses into corresponding -वाला  phrases:
 Models:
 a.  वह ऐसी क़मीज़ चाहती है जिस में जेबें हों।     =>
         वह जेबों वाली क़मीज़ चाहती है।
 b.  जो  ( लोग )  रो रहे हैं वह क्यों रो रहे हैं ?     =>
         रोनेवाले क्यों रो रहे हैं ?
  1.  हम उस मकान में रहते हैं जो सामने है।       =>
  2.  क्या नारियल उस  ( अादमी )  के पास मिलते हैं जो फल  ( बेचता है  ) ?       =>
  3.  अाप उस गाड़ी से जाइये जो सात बजे  ( छूटती है )  ।      =>
  4.  माचिस उस दुकान में नहीं मिलेगी जहाँ किताबें  ( बेचते हैं )  ।      =>
  5.  हमारा साहब एक ऐसा अादमी है जिसके पास  ( बहुत )  पैसा है।      =>
  6.  इस तरह के बिस्कुट उन  ( अादमियों )  के पास नहीं मिलेंगे जो चाय  ( बेचते हैं  )      =>
  7.  जो तसवीर बाएँ हाथ पर है वह मुझे सचमुच पसन्द अाई।       =>
  8.  जो  ( लोग )  पैसे माँगते हैं उन्हें भिखारी कहते हैं।      =>
  9.  उस  ( अादमी )  से क्या मिलेगा जिसकी जेबें  ( हमेशा  )  ख़ाली ( रहती हैं  ) ?       =>
10.  जो घोड़े ताँगे  ( खींचते हैं )  वह बहुत तेज़ नहीं भाग सकते।      =>
