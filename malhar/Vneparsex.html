Exercise on V- ने पर and V- ने से.

   Using the expressions given in parentheses, respond to the questions or statements.

Model 1:    क्या तुमने मेरा लिफ़ाफ़ा भेज दिया ?       ( रह जाना )          =>
                  नहीं,    मैं तुम्हारा लिफ़ाफ़ा /   भेजने से / /   भेजते - भेजते /   रह गया।

Model 2:   अबे,  तूने सिगरेट पीना बन्द किया ?       ( घरवाली,  मजबूर करना )           =>
                  हाँ,   घरवाली ने मुझे बन्द करने /  पर / /  के लिये /  मजबूर कर दिया।

1.    कहानी अच्छी है, अाप उसे छपा रही हैं ?        ( पति देव;  मना करना )       =>

2.    ठीक है,  तुम ख़ुश हो।  लेकिन दादा ?        ( जान लेना;  उतारू होना )       =>

3.   मैं डैडी से तुम्हारे कुकर्म कहूँगी।     ( नाक कटाना;  लगी हुई होना )       =>

4.   लगता है,  तुम्हारी पत्नी ने बड़ी महँगी साड़ी ख़रीदी है।        ( सहेलियाँ,  उकसा देना )       =>

5.  ( बस में मत चढ़ो ! )   इस भीड़ में तुम्हारी जेब कटेगी !       ( V- ने का डर )        =>

6.  ये डाकू अब हमारा सारा माल मारकर ही रहेंगे।        ( कौन;   रोकना )       =>

7.  उनके बच्चे कैसे हैं ?        ( मेज़ पर चढ़ना;   बाज़ न अाना )       =>

8.  मज़दूर लोग क्यों नहीं अाए ?        ( काम करना;   इनकार कर देना )       =>
