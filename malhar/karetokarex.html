यूनीवर्सिटी आॅफ मिशिगन

'Only X can / will do it!'

Exercise on X ही करे ताे करे

Rewrite as X  ही करे ताे करे structures. When possible use derived intransitives or passives in the second clause. Translate.

Model 1:   यह मकान सिर्फ तुम्हीं बना सकाेगे।  हमारे ताे बूते के बाहर है।     =>
                 यह मकान तुम्हीं बनाअाेगे ताे बनेगा।  हमारे ताे बूते के बाहर है।
                 'Only you can build this house. It's beyond us.'

Model 2:   अब ख़ुदा ही मुझे उबारें।     =>
                 अब ख़ुदा ही मुझे उबारें ताे मैं /   उबरूँ / /   उबर जाऊँ /  ।
                 'Only God can save me now.'

1.  सिर्फ तुम्हीं यह दरवाज़ा खाेल सकाेगे।     =>
2.  अाजकल केवल निरजंन ही मिलता है।  बाक़ी सब घर में छिपकर रहते हैं।     =>
3.  सिर्फ " निरमा "  गन्दगी काे निकाल सकेगी।  वरना सफेदी अाने का नाम नहीं लेती।     =>
4.  बस तू ही मदद करने अाया।  अाैर दाेस्त ग़ायब !     =>
5.  सिर्फ मैं ही लड़ाई करने जाऊँ।  बाक़ी सब कायर जाे हैं !     =>
6.  यह रक़म केवल रईस लाेग देंगे।  हमसे ताे नहीं दी जाएगी।     =>
