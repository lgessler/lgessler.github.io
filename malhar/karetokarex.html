यूनीवर्सिटी आॅफ़ मिशिगन

'Only X can / will do it!'

Exercise on X ही करे तो करे

Rewrite as X  ही करे तो करे structures. When possible use derived intransitives or passives in the second clause. Translate.
Model 1:   यह मकान सिफऱ् तुम्हीं बना सकोगे।  हमारे तो बूते के बाहर है।     =>
                 यह मकान तुम्हीं बनाओगे तो बनेगा।  हमारे तो बूते के बाहर है।
                 'Only you can build this house. It's beyond us.'
Model 2:   अब ख़ुदा ही मुझे उबारें।     =>
                 अब ख़ुदा ही मुझे उबारें तो मैं /   उबरूँ / /   उबर जाऊँ /  ।
                 'Only God can save me now.'
1.  सिफऱ् तुम्हीं यह दरवाज़ा खोल सकोगे।     =>
2.  आजकल केवल निरजंन ही मिलता है।  बाक़ी सब घर में छिपकर रहते हैं।     =>
3.  सिफऱ् " निरमा "  गन्दगी को निकाल सकेगी।  वरना सफ़ेदी आने का नाम नहीं लेती।     =>
4.  बस तू ही मदद करने आया।  औरर दोस्त ग़ायब !     =>
5.  सिफऱ् मैं ही लड़ाई करने जाऊँ।  बाक़ी सब कायर जो हैं !     =>
6.  यह रक़म केवल रईस लोग देंगे।  हमसे तो नहीं दी जाएगी।     =>
